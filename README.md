## 📌 Description
A fully responsive business website built for CCTV installation and security services. This project demonstrates frontend development skills including layout design, responsiveness, and UI structuring using Bootstrap.

## 🚀 Features
- Fully responsive design
- Service showcase section
- Contact section
- Modern UI using Bootstrap

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap
- JavaScript
